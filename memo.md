# メモ
## 基本
- 要素数が多い配列は大域変数として宣言する
10^6個要素がある配列をmain関数内で宣言したらsefmentation faultになった。
大域変数で宣言したら問題なく動いた。

## 標準ライブラリ(STL)
### map
連想配列的なもの
- 宣言
`map<(型1), (型2)> (変数名)`  
型1: 添え字の型  
型2: 入れるデータの型  
- アクセス
配列と同じ  
ex) `map<int, int> a; a[0] = 0;`
- (入れるデータの型がintの時)要素が入っていないときは0を返す

### set
集合(重複なし)
- 宣言  
`set<(型)> (変数名)`
- 全要素探索
```
set<int> Set
for(auto itr = Set.begin(); itr != Set.end(); itr++) {
    cout << (*itr) << endl;
}
```

### vector
- 初期化
```
vector<int> a(4, 5) //要素数4、値5で初期化
```
